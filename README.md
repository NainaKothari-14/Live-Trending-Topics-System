# Live Trending Topics System

A real-time system to track trending topics using Node.js, Socket.IO, Redis, and Elasticsearch.

## Features
- Real-time updates of trending topics
- Fast analytics and search
- Persistent storage with MongoDB

## Tech Stack
Node.js | Socket.IO | Redis | Elasticsearch | MongoDB

## How to Run
1. Clone the repo
2. Install dependencies: `npm install`
3. Start the server: `npm start`
